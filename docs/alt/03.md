# Настройка VLAN

* Создать каталог для VLAN и файл **/etc/net/ifaces/enp2s0.270/options**
```
BOOTPROTO=static
CONFIG_WIRELESS=no
CONFIG_IPV4=yes
HOST=enp2s0
ONBOOT=yes
TYPE=vlan
VID=270
```

* Настроить Ethernet-мост **/etc/net/ifaces/vmbr1/options**
```
BOOTPROTO=static
CONFIG_WIRELESS=no
CONFIG_IPV4=yes
HOST='enp2s0.270'
ONBOOT=yes
TYPE=bri
```

* IP-адрес моста **/etc/net/ifaces/vmbro/ipv4address**
```
192.168.5.141/24
```

* Маршрут моста **/etc/net/ifaces/vmbro/ipv4route**
```
default via 192.168.5.250
```
